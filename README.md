# Sahal Tech — Badge STIC

Frontend React (Vite) pour la réservation de badges, vérification et scanner d'entrée.

## Développement

```bash
cp .env.example .env
npm install
npm run dev
```

## Production

L'API Laravel tourne sur le VPS : `http://13.48.134.87:8090/api`

```bash
npm run build
```

Le build utilise `.env.production`.
